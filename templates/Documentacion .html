Documentación de la API REST

Obtener todos los contactos

URL: /contacts

Método: GET

Descripción: Devuelve todos los contactos almacenados en la base de datos.

Respuesta exitosa: Código de estado 200 (OK)

{
    "contacts": [
        {
            "id": 1,
            "name": "John Doe",
            "email": "john@example.com",
            "phone": "1234567890"
        },
        {
            "id": 2,
            "name": "Jane Smith",
            "email": "jane@example.com",
            "phone": "9876543210"
        }
    ]
}

Crear un nuevo contacto

URL: /contacts

Método: POST

Descripción: Crea un nuevo contacto en la base de datos.

Cuerpo de la solicitud:

{
    "name": "Nombre del contacto",
    "email": "Correo electrónico del contacto",
    "phone": "Número de teléfono del contacto"
}

Respuesta exitosa: Código de estado 201 (Creado)

{
    "message": "Contacto creado con éxito",
    "contact": {
        "id": 3,
        "name": "Nuevo Contacto",
        "email": "contacto@example.com",
        "phone": "5555555555"
    }
}

Obtener un contacto específico

URL: /contacts/<id>

Método: GET

Parámetro de ruta: id (entero) - ID del contacto

Descripción: Obtiene los detalles de un contacto específico.

Respuesta exitosa: Código de estado 200 (OK)

{
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com",
    "phone": "1234567890"
}

Actualizar un contacto existente

URL: /contacts/<id>

Método: PUT o PATCH

Parámetro de ruta: id (entero) - ID del contacto
